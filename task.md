## 🛡️ Strict Quality & Batch Protocol (The Batch-of-5 Golden Rule)
To prevent rate-limiting and ensure pristine artistic quality, all generation MUST follow this process without exception:
1. **Paced Generation**: Launch generation command restricted to a limit of 5 cards: `python scripts/generate_assets.py --level <level> --limit 5`
2. **Copy to Artifacts**: Copy the 5 generated `.webp` files from `<level>/images/` to the conversation artifacts directory (`C:\Users\aman-\.gemini\antigravity\brain\c2a5c57a-f6ca-42ed-a378-ce7fe7c36a6b/`).
3. **Visual Quality Audit**: Open and inspect every single one of the 5 images using the native `view_file` tool to manually check them with honesty!
4. **Intuitiveness Check**: Verify that the generated 3D visual clearly and intuitively depicts the German word's meaning. If abstract, buggy, or sub-optimal, delete and run a targeted regeneration.
5. **Progressive Commit**: Once verified, stage and commit the 5 images with semantic message: `feat(assets): generate and audit <level> cards <start>-<end>`
6. **Documentation Sync**: Append the results to `walkthrough.md` using a beautiful markdown carousel, and mark off the checklist below.
